# Text_Storer

Simple application using <b>GTKmm</b> and <b>SQLite3</b>.</br>

## Test

- Debian 9

## Requirements

- libgtkmm-3.0-dev
- sqlite3
- gcc-6

## References
[GTKmm](https://gtkmm.org/en/) : C++ Interfaces for GTK+ and GNOME.</br>
[SQLite](https://www.sqlite.org/) : SQL database engine.</br>
